### ⚙️ &nbsp;[RuntimeX](https://github.com/vmore2/runtimex)

> *"Debugging an AI agent shouldn't feel like forensics."*

A **model-agnostic AI agent harness** solving the black-box problem in LLM workflows. Built for complete observability.

- 🔁 **Deterministic Replay** — zero API cost
- ⏪ **Time-Travel Fork** — edit step N, go live from N+1
- 🧠 **Failure Memory** — self-healing agents
- 📡 Event log: Pydantic → SQLite → OpenTelemetry

```python
from runtimex import Agent, LiteLLMModel
agent = Agent(model=LiteLLMModel("gpt-4o-mini"))
result = agent.run_sync("Summarise CHANGELOG.md")
```
