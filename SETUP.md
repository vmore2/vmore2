# GitHub Profile README
This folder contains the source for vmore2's GitHub profile (https://github.com/vmore2/vmore2).

## Setup
1. Create a new GitHub repository named exactly `vmore2` (same as username)
2. Push this README.md to it — GitHub will auto-display it on your profile.

## Deploy
```bash
git init
git add README.md
git commit -m "Add GitHub profile README"
git branch -M main
git remote add origin https://github.com/vmore2/vmore2.git
git push -u origin main
```
